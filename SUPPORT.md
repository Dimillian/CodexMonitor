# Support

## Getting Help

1. Check `README.md` for setup and run commands.
2. Check `docs/codebase-map.md` for task-to-file navigation.
3. Check domain runbooks under `docs/testing`, `docs/ci-cd`, `docs/debug`, and `docs/ui-ux`.

## Bug Reports

When reporting issues, include:

1. Environment (OS, Node, Rust, app version)
2. Exact reproduction steps
3. Logs/errors/screenshots
4. Expected vs actual behavior
