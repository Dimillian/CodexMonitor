# Documentation Governance Checklist

Use this checklist before merging:

1. Contracts updated?
   - `AGENTS.md`
   - `CLAUDE.md`
   - `docs/ai/agent-contract.md`
2. Navigation updated?
   - `docs/codebase-map.md`
   - root `README.md` doc links
3. Domain docs updated for touched areas?
   - `docs/testing/*`
   - `docs/ci-cd/*`
   - `docs/debug/*`
   - `docs/ui-ux/*`
4. CI doc gates passing?
   - `doc-drift`
   - docs integrity job
